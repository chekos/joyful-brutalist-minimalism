# Joyful Brutalist Minimalism

Joyful Brutalist Minimalism is Sergio's authored design language for the web:
documented in prose, explored in Figma, and proven through coded specimens.

It is a design practice, not a framework that every project must import. Each
site can interpret the language according to its content and purpose while
sharing a recognizable way of thinking: honest structure, useful delight,
editorial warmth, technical precision, and inspectable behavior.

## Project forms

- `DESIGN.md` is the agent-readable design constitution: principles, judgment,
  vocabulary, and boundaries.
- [Joyful Brutalist Minimalism in Figma](https://www.figma.com/design/T4jEmsyQBURKMr6s3zYfFJ)
  is the visual laboratory for foundations, components, patterns, and studies.
- The reference site will be the executable specimen: responsive layout,
  interaction, motion, accessibility, and browser behavior.
- Platform-neutral tokens will bridge visual decisions and code without making
  consumer projects depend on one component implementation.
